# Surfer-s-PCB

A blinky pad that has written on it one of my favorite quotes.

I didn't really know what to do with the design so I added a little comic book reference.

The LEDs are lined up in the shape of a sephiroth tree because I thought it would be cool

# Why I made this

I designed this project as a piece of tech-art. Rather than a standard grid, the LEDs are arranged in the shape of the **Sephiroth Tree (Tree of Life)** to create a striking visual effect when powered on. 

It’s meant to be framed or kept on a desk as a conversation piece.

2. **Assemble:** Solder the LEDs and resistors to the designated pads.
3. **Display:** Power it up and enjoy the glow!
